### 关于读书与笔记：       

遇到精彩地方可以记录下来，切忌直接大段的摘抄。       



**笔记要求：**     

* 为什么要记录
* 最好附上代码例子
* 第一遍看书坚决不记录



**反思：**       

以前自己的笔记大多是大段的摘抄，还有就是记些上下文关联不大的，总之就是非常不好。     

今后的笔记需要对整本书的结构理解后，再思考后记录。        





**关于读书：**         

看书不要急切，最好一个章节一个章节的挨着看。不要大段的摘抄了，自己都不会看的。       



总之就是读书要慢，总结不是摘抄。记录的话至少在第二遍（会看两遍的大概不是差书吧）
